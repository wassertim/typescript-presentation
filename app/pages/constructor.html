Constructor

  Typescript
  
  class Greeter {
      static standardGreeting = "Hello, there";
      greeting: string;
      greet() {
          if (this.greeting) {
              return "Hello, " + this.greeting;
          }
          else {
              return Greeter.standardGreeting;
          }
      }
  }

The constructor function is always there
